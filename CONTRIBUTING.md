# Contributing

Thank you for helping improve Vibe Coding Prompt Workflow.

## Issues And Suggestions

Issues are welcome for:

- unclear instructions or missing edge cases;
- installation or compatibility problems;
- realistic examples and forward-testing results;
- source corrections and accessibility improvements.

When reporting a behavior problem, include the original request, the generated executable prompt, the model or agent used, and the observed result. Remove secrets and personal information first.

## Pull Requests

Unsolicited pull requests are not accepted at this stage. This project is distributed under CC BY-NC-ND 4.0, and adapted versions may not be distributed without prior written permission.

Open an Issue before proposing a change. The author may implement the suggestion directly or explicitly invite a contribution and agree on the required rights and attribution.
